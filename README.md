# VNote_Winapp_Jiaen
資展國際第一次個人 Winform 專題。

## 說明
本專案開發使用 C# / YouTube Data API / 連接至 SQLServer。
能夠進行使用者資料以及筆記內容的 資料增刪查改。
使用三層式架構與關聯式資料表設計，增加系統未來的可擴展性與可維護性，增加系統的整題效率
